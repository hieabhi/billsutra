# BillSutra Desktop Icon Placeholder

Place your application icon here:
- Windows: icon.ico (256x256 or higher)
- Mac: icon.icns
- Linux: icon.png (512x512 recommended)

For now, a default icon will be used during development.

You can create icons using:
- https://www.icoconverter.com/ (free online tool)
- https://iconverticons.com/online/ (converts PNG to ICO/ICNS)

Recommended sizes:
- 16x16, 32x32, 48x48, 64x64, 128x128, 256x256, 512x512
